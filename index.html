No Masters Project

No Masters is a project of a global network of mutual support for the strengthening of self-organization that aims to create a sustainable ecosystem based on win-win relationships.

Through empowerment, direct action and encouraging autonomy, this project aims to establish a collaborative network that facilitates the organization and communication of individuals, groups, bands, movements and organizations with libertarian and DIY related principles.
